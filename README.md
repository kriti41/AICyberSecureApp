# AICyberSecureApp
AI CyberSecure App is a **multi-module cybersecurity tool** that leverages AI and Machine Learning to detect phishing, malware, ransomware behavior, intrusion, and brute-force attacks.

##  Features

1. **Phishing URL Detection**  
   - Detects suspicious URLs using trained AI models.  
   - Input via manual URL entry or CSV upload.  

2. **Malware Detection (Static Features)**  
   - Predicts malware presence from file static features.  

3. **Ransomware Behavior Detection**  
   - Detects ransomware behavior based on system activity patterns.  

4. **Intrusion Detection System (IDS)**  
   - Uses NSL-KDD dataset features to predict normal vs attack network traffic.  

5. **Brute Force Attack Detection**  
   - Detects suspicious login attempts using password attempts, login success, and timestamp features.

##  Goal / Vision

The vision of AI CyberSecure App is to transform into an **extensible, all-in-one cybersecurity platform** that leverages AI to detect, prevent, and mitigate a wide range of cyber threats in real-time.  

Future goals include:  
- Adding support for **cloud-based threat monitoring**  
- Incorporating **real-time network anomaly detection**  
- Extending to **enterprise-level security analytics**  
- Building a **community-driven plugin system** for AI-based cybersecurity tools
